## Execution method if you don't want install global gem

1. do git clone

2. Adjust ruby version (2.5.0)

3. `$ gem install bundler`

4. `$ bundle install --path vendor/bundle`

5. `$ bundle exec rails db:migrate`

6. `$ bundle exec rails s`
